# Chrome Extension Icons

This directory contains PNG icons for the Chrome extension identity. The extension no longer exposes a toolbar popup; all user interaction is handled by the page-side control drawer injected by `content.js`.

## Files

- `icon16.png`: toolbar and compact UI icon.
- `icon48.png`: extension management page icon.
- `icon128.png`: high-resolution extension icon.

## Updating Icons

Use `icons/export.html` to regenerate PNG assets, then reload the extension.
