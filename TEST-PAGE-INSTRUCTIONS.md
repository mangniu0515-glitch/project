# 二维码测试页面使用说明

## 📍 访问地址

测试页面已启动在：**http://localhost:8080/test-qrcode.html**

## ✨ 功能特性

### 1. 动态生成二维码
- 输入任意文本、URL、数字等
- 点击"生成二维码"按钮即时生成
- 页面加载时自动生成示例二维码

### 2. 美观的界面
- 渐变紫色背景
- 圆角卡片设计
- 平滑动画效果
- 响应式布局

### 3. 使用提示
页面底部提供了详细的使用说明：
- 在文本框中输入任意内容
- 点击"生成二维码"按钮生成二维码
- 使用 **二维码采集器插件** 扫描测试

## 🔌 配合插件测试

### 测试步骤

1. **打开测试页面**
   ```
   访问 http://localhost:8080/test-qrcode.html
   ```

2. **生成测试二维码**
   - 默认已有一个示例二维码
   - 可修改文本内容生成不同二维码
   - 支持：URL、文本、数字、邮箱等

3. **安装插件**（如未安装）
   - 打开 Chrome：`chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择：`d:/qrcode-collection-system/chrome-extension/`

4. **使用插件扫描**
   - 打开包含二维码的网页（本测试页面）
   - 点击 Chrome 工具栏中的插件图标
   - 点击"扫描当前页面"按钮
   - 插件会自动识别页面中的二维码

5. **查看结果**
   - 扫描结果会显示在插件的历史记录中
   - 可点击"上传待发"将数据上传到服务器
   - 上传后可在管理后台查看

## 🎯 测试建议

### 测试不同类型的内容：
1. **URL链接**：如 `https://www.example.com`
2. **纯文本**：如 `Hello, World!`
3. **数字**：如 `123456789`
4. **中文内容**：如 `二维码测试`
5. **复杂内容**：如 `BEGIN:VCARD...` (联系人信息)

### 测试场景：
1. **单个二维码**：页面上显示一个二维码
2. **多个二维码**：复制多个二维码图片到页面
3. **动态二维码**：实时修改内容生成新二维码
4. **清除重试**：点击"清空"后重新生成
