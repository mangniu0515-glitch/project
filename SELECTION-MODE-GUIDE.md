# 二维码采集器 - 选取模式使用指南

## 🎯 核心功能：半自动化元素选取

新的选取模式让您可以精准控制插件监控的页面区域，无需扫描整个页面。

---

## 📖 使用步骤

### 1️⃣ 打开插件面板
点击 Chrome 工具栏中的插件图标，打开插件面板

### 2️⃣ 点击"选取模式"按钮
在插件面板中找到 **"🎯 选取模式"** 按钮并点击

### 3️⃣ 进入选取状态
- 页面会出现十字光标
- 鼠标移动到页面元素上时，元素会被绿色边框高亮显示
- 底部右侧会弹出预览面板

### 4️⃣ 选择目标元素
点击您想要监控的页面元素（可以是 div、section、容器等）

### 5️⃣ 预览与确认
系统会：
- 显示选择的元素信息
- **智能检测**该元素内是否包含二维码
- 给出预览结果

#### 如果发现二维码：
点击 **"绑定并开始监控"** 按钮

#### 如果未发现二维码：
也可以点击 **"绑定元素"** 按钮，之后如果该元素内出现二维码会及时通知您

### 6️⃣ 绑定成功
- 插件会显示 **"✅ 二维码监控已启用"** 提示
- 页面右上角会出现绿色徽章，表示监控已启动
- 以后打开该网页时，插件会自动定位并监控该元素

---

## ✨ 功能特性

### 🕵️ 智能预览
- **实时检测**：鼠标悬停时自动检测元素内的图片
- **二维码识别**：使用 jsQR 库识别二维码内容
- **选择器生成**：自动生成简洁的 CSS 选择器

### 🔗 持久绑定
- **按域名存储**：每个网站独立保存选择器
- **自动恢复**：下次访问同一网站时自动加载绑定
- **轻松解除**：点击右上角徽章即可解除绑定

### 🔔 冒泡通知
发现二维码时：
- 在二维码图片旁边显示**冒泡通知**
- 紫色渐变背景，美观醒目
- 显示二维码内容预览
- **点击即可添加到上传队列**

### ⚙️ 灵活配置
- **自动上传开关**：可选择是否自动上传
- **扫描间隔**：可调整检测频率（1-60秒）
- **服务器配置**：设置上传地址和 API 密钥

## 💡 使用场景

### 场景 1：固定二维码区域
**情况**：网站有一个专门展示二维码的区域

**操作**：
1. 点击"选取模式"
2. 点击二维码所在区域
3. 绑定该容器元素
4. 之后自动监控该区域

### 场景 2：动态二维码
**情况**：二维码会动态加载或更新

**操作**：
1. 点击"选取模式"
2. 选择包含二维码的容器
3. 绑定后设置较短的扫描间隔（如3秒）
4. 每次二维码更新都会收到通知

### 场景 3：批量监控多个区域
**情况**：一个页面有多个需要监控的区域

**操作**：
- 每个域名只能绑定一个选择器
- 可以先绑定大容器，再通过 CSS 选择器精确控制
- 或在不同子页面分别绑定

## 🔧 技术细节

### 选择器生成策略
优先级：
1. **ID 选择器**：如果有 `id` 属性
2. **Class 选择器**：如果有 `class` 属性
3. **层级选择器**：使用 `:nth-child()` 定位

### 存储结构
```javascript
{
  boundSelectors: {
    "https://example.com": "#qr-container",
    "https://test.com": "div.qr-wrapper"
  }
}
```

### 监控机制
- 使用 `setInterval` 定期扫描
- 已处理的图片会缓存，避免重复识别
- 每次扫描独立执行，不互相干扰

---

## ⚠️ 注意事项

1. **跨域限制**
   - 选择器只能在本域名内有效
   - 不同域名需要分别绑定

2. **元素存在性**
   - 如果绑定的元素不存在，会自动清除绑定
   - 下次访问时需要重新绑定

3. **性能考虑**
   - 扫描间隔不宜过短（建议 3-5 秒）
   - 绑定的元素不宜过大
   - 已处理的图片不会重复扫描

4. **网络要求**
   - 需要配置服务器地址才能上传
   - 确保 API 密钥正确

## 🚀 快速开始

1. **打开插件面板**
   - 点击工具栏插件图标

2. **配置服务器**（首次使用）
   - 点击"⚙️ 设置"
   - 填写服务器地址和 API 密钥
   - 点击"保存"

3. **进入选取模式**
   - 点击"🎯 选取模式"按钮

4. **选择目标元素**
   - 在页面上移动鼠标，观察高亮效果
   - 点击要监控的元素

5. **确认绑定**
   - 查看预览面板的检测结果
   - 点击"绑定并开始监控"

6. **自动监控**
   - 关闭插件面板
   - 系统自动监控该元素
   - 发现二维码时点击冒泡通知

## 📞 故障排除

### Q: 点击"选取模式"没反应？
**A**: 确保当前页面可以执行脚本（不是 chrome:// 页面）

### Q: 预览面板没有出现？
**A**: 检查是否有 JavaScript 错误，尝试刷新页面

### Q: 绑定后没有收到通知？
**A**: 
- 检查元素内确实有二维码图片
- 确认自动上传开关已开启
- 尝试缩短扫描间隔

### Q: 如何更换绑定的元素？
**A**: 再次点击"选取模式"，重新选择新元素即可覆盖

### Q: 如何完全停止监控？
**A**: 
- 点击右上角的绿色徽章
- 或在插件面板中再次点击"选取模式"
